# 像素图生成器 - 拼豆图纸工具

一个轻量级的像素图生成工具，专为拼豆（Perler bead）手作创作者设计。可以将任意图片转换为像素图纸，支持实时编辑、颜色匹配和图纸库管理。

## 功能特性

- 🎨 **图片像素化**：将上传的图片转换为指定尺寸的像素网格
- 🎯 **颜色匹配**：自动匹配拼豆色号，支持 MARD 品牌色板（264色）
- ✏️ **实时编辑**：支持手动编辑图纸，包括画笔、橡皮擦、取色器等功能
- 📚 **图纸库管理**：保存、编辑、下载历史图纸
- 📱 **响应式设计**：适配各种设备屏幕

## 技术栈

- 纯前端实现（HTML + CSS + JavaScript）
- Tailwind CSS（UI 框架）
- HTML5 Canvas（图像处理和绘制）
- LocalStorage（本地数据存储）

## 使用说明

### 1. 开始创作
- 在首页点击"开始创作"
- 选择色板（目前支持：拼豆 - MARD）
- 选择画板尺寸（52×52 或 104×104）
- 上传图片并调整缩放/位置
- 点击"生成图纸"

### 2. 编辑图纸
- 生成后自动进入编辑模式
- 使用工具栏进行修改：
  - **画笔**：选择颜色绘制
  - **橡皮擦**：清除颜色
  - **取色器**：从图纸中选取颜色
  - **缩放**：放大缩小视图
  - **撤销/重做**：支持操作历史

### 3. 保存与管理
- 点击"保存为新图纸"保存到图纸库
- 从图纸库可以重新加载编辑
- 支持下载高分辨率图纸

## 部署说明

### GitHub Pages 部署

1. 在 GitHub 创建新仓库
2. 将代码推送到仓库
3. 进入仓库 Settings → Pages
4. Source 选择分支（如 `main`）
5. 几分钟后访问：`https://你的用户名.github.io/仓库名/`

### 其他部署方式

- **Vercel**：导入 GitHub 仓库自动部署
- **Netlify**：拖拽项目文件夹或连接 GitHub
- **其他静态托管服务**：上传项目文件即可

## 注意事项

- 所有数据存储在浏览器 LocalStorage 中，清除浏览器数据会导致数据丢失
- 建议定期备份重要的图纸数据
- 色板数据位于 `colorPalette.js`，可根据需要添加更多颜色

## 开发说明

本项目为纯前端应用，无需服务器或数据库。所有功能均运行在浏览器端。
